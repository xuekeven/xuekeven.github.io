JavaScript语言基础之语句

ECMA-262 描述了一些语句（也称为流控制语句），而 ECMAScript 大部分语法都体现在语句中。语句通常使用一或多个关键字完成既定的任务。语句可以简单，也可以复杂。简单的如告诉函数退出，复杂的如列出一堆要重复执行的指令。

if语句

if 语句是使用最频繁的语句之一，语法如下：
if (condition) statement1 else statement2

其中的条件（condition）可以是任何表达式，并且求值结果不一定是布尔值。ECMAScript 会自动调用 Boolean() 函数将这个表达式的值转换为布尔值。如果条件求值为 true，则执行语句 statement1；如果条件求值为 false，则执行语句 statement2。这里的语句可能是一行代码，也可能是一个代码块（即包含在一对花括号中的多行代码）。

连续使用多个if语句：
if (condition1) statement1 else if (condition2) statement2 else statement3
