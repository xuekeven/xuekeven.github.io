CSS基础

  • 基础

    • 显示类型
      • display
        • display 属性可以设置元素的内部和外部显示类型。元素的外部显示类型将会决定该元素在流式布局中的表现；元素的内部显示类型可以控制其子元素的布局
        • display 属性使用关键字取值来指定，关键字取值被分为六类
        • display-outside（外部值）
          • 这些关键字指定了元素的外部显示类型
          • block：这个值会生成一个块级元素盒子，该元素之前和之后打断（换行）
          • inline：这个值会生成一个行内元素盒子，该元素之前和之后不会打断（换行）。如果空间充足，该元素后的元素将会在同一行显示
          • run-in：这个值生成一个 run-in 盒子。如果其后的兄弟元素是块级盒子，那么此 run-in 盒子就会以兄弟元素的第一个行内元素盒子的形式存在；如果其后的兄弟元素是行内元素，那么这个run-in盒子会变成块级元素。run-in元素的存在形式取决于相邻元素，如果它的内部存在 block box，那么它会变成块级元素盒子
        • display-inside（内部值）
          • 这些关键字指定了元素的内部显示类型
          • flow：目前含义不清，实验室阶段产品
          • flow-root：可以让元素块状化，同时包含 BFC，可以用来清除浮动，去除 margin 合并，实现两栏自适应布局等
          • table：此元素作为块级表格来显示（类似 <table>），表格前后带有换行符
          • flex：此元素作为一个弹性布局盒子
          • grid：此元素作为一个网格布局盒子
          • ruby：此元素可以显示旁注标记，与 ruby 标签配合使用
        • display-listitem（列表值）
          • 将这个元素的外部显示类型变为 block 盒，内部显示类型变为多个 list-item inline 盒
          • list-item：此元素作为列表显示
        • display-internal（属性值）
          • 有些布局模型（如 table 和 ruby）有着复杂的内部结构，因此它们的子元素可能扮演着不同的角色。这一类关键字就是用来定义这些“内部”显示类型，并且只有在这些特定的布局模型中才有意义
          • table-row-group：此元素作为一个或多个行的分组来显示（类似 <tbody>）
          • table-header-group：此元素作为一个或多个行的分组显示（类似 <thead>）
          • table-footer-group：此元素作为一个或多个行的分组显示（类似 <tfoot>）
          • table-row：此元素作为一个表格行显示（类似 <tr>）
          • table-cell：此元素作为一个表格单元格显示（类似 <td> 和 <th>）
          • table-column-group：此元素作为一个或多个列的分组显示（类似 <colgroup>）
          • table-column：此元素作为一个单元格列显示（类似 <col>）
          • table-caption：此元素作为一个表格标题显示（类似 <caption>）
          • ruby-base：此元素的行为类似于 <rb>
          • ruby-text：此元素的行为类似于 <rt>
          • ruby-base-container：此元素的行为类似于 <rbc>
          • ruby-text-container：此元素的行为类似 <rtc>
        • display-box（显示值）
          • 这些值决定元素是否使用盒模型
          • none：关闭此元素的显示，不影响布局（文件中没有该元素）。所有子项的显示也被关闭。要一个元素占据空间（文件中存在），但不渲染，使用 visibility 属性
          • contents：此元素自己不显示。它们被它们的伪盒和子项盒取代
        • display-legacy（混合值）
          • CSS Level 3 规范详细说明了 display 属性的两类取值——显式地指定了外部和内部显示属性的规范——但是还没有被浏览器广泛支持
          • inline-block：此元素会产生一个块元素盒子，并且像内联盒子一样（表现得更像一个被替换的元素），可以融入到周围内容中
          • inline-table：此元素作为内联表格显示（类似 <table>），表现为一个不同于块级盒子的内联盒子。表盒子内部是一个块级上下文
          • inline-flex：元素表现为一个内联元素，并对内容采用弹性盒子模型进行布局
          • inline-grid：元素表现为一个内联元素，并对内容采用网格模型进行布局
    • 可见性
      • visibility
        • visible：元素是可见的。默认值。
        • hidden：元素是不可见的
        • collapse：当在表格元素中使用时，此值可删除一行或一列，但它不影响表格布局。它被行或列占据的空间会留给其他内容使用。如果此值被用在其他的元素上会呈现为 “hidden”
    • 高度
      • height
        • 不显式设置，浏览器会使用 auto
      • min-height
        • 属性指定最小或最大值，而不是明确定义高度，元素就可以在这些界限内自动决定高度
      • max-height
        • 属性指定最小或最大值，而不是明确定义高度，元素就可以在这些界限内自动决定高度
        • 在小区域上会填满屏幕，在大区域上它会扩展到最大值。相比设置 height，这种方式能够有效避免在小区域上出现垂直滚动条
    • 宽度
      • width
        • 不显式设置，浏览器会使用 auto
      • min-width
        • 属性指定最小或最大值，而不是明确定义宽度，元素就可以在这些界限内自动决定宽度
      • max-width
        • 属性指定最小或最大值，而不是明确定义宽度，元素就可以在这些界限内自动决定宽度
        • 在小视口上会填满屏幕，在大视口上它会扩展到最大值。相比设置 width，这种方式能够有效避免在小屏幕上出现水平滚动条
    • 内边距
      • padding
        • 简记法，同时设置四个边的内边距。如果需要，输入 -top、-right、-bottom、-left 将边框效果限制在某一条边上
          • padding-top
          • padding-right
          • padding-bottom
          • padding-left
        • padding 值为百分比时，其值是基于其父元素的宽度（width，不包含 padding 和 border）来定义的
    • 外边距
      • margin
        • 简记法，同时设置四个边的内边距。如果需要，输入 -top、-right、-bottom、-left 将边框效果限制在某一条边上
          • margin-top
          • margin-right
          • margin-bottom
          • margin-left
    • 水平对齐
      • text-align
        • 定义行内元素的内容如何相对它的块父元素对齐
        • left：行内内容向左侧边对齐。默认值由浏览器决定。
        • right：行内内容向右侧边对齐
        • center：行内内容居中
        • justify：文字向两侧对齐，对最后一行无效
        • justify-all：和 justify 一致，但是强制使最后一行两端对齐
    • 垂直对齐
      • vertical-align
        • 指定行内元素（inline）或表格单元格元素（table-cell）的垂直对齐方式
        • 行内元素的值
          • baseline：使元素的基线与父元素的基线对齐。默认值。
          • sub：使元素的基线与父元素的下标基线对齐
          • super：使元素的基线与父元素的上标基线对齐
          • text-top：使元素的顶部与父元素的字体顶部对齐
          • text-bottom：使元素的底部与父元素的字体底部对齐
          • top：使元素及其后代元素的顶部与整行的顶部对齐
          • middle：把此元素放置在父元素的中部
          • bottom：使元素及其后代元素的底部与整行的底部对齐
          • <length>：将元素升高或降低指定的高度，可以是负数
          • %：使用 “line-height” 属性的百分比值来排列此元素，允许使用负值
        • 表格单元格元素的值
          • baseline：使单元格的基线，与该行中所有以基线对齐的其它单元格的基线对齐
          • top：使单元格内边距的上边缘与该行顶部对齐
          • middle：使单元格内边距盒模型在该行内居中对齐
          • bottom：使单元格内边距的下边缘与该行底部对齐
    • 处理溢出
      • overflow
        • visible：内容不会被修剪，让元素盒子中的所有内容可见。默认值。
        • hidden：裁剪内容 - 不提供滚动机制
        • scroll：裁剪内容 - 提供滚动机制
        • auto：让滚动条仅在访问者访问溢出内容时出现
      • overflow-x
        • visible：内容不会被修剪，让元素盒子中的所有内容可见。默认值。
        • hidden：裁剪内容 - 不提供滚动机制
        • scroll：裁剪内容 - 提供滚动机制
        • auto：让滚动条仅在访问者访问溢出内容时出现
        • no-display：如果内容不适合内容框，则删除整个框
        • no-content：如果内容不适合内容框，则隐藏整个内容
      • overflow-y
        • visible：内容不会被修剪，让元素盒子中的所有内容可见。默认值。
        • hidden：裁剪内容 - 不提供滚动机制
        • scroll：裁剪内容 - 提供滚动机制
        • auto：让滚动条仅在访问者访问溢出内容时出现
        • no-display：如果内容不适合内容框，则删除整个框
        • no-content：如果内容不适合内容框，则隐藏整个内容
    • 指针形状
      • cursor
        • default：默认光标
        • crosshair：光标呈现为十字线
        • move：此光标指示某对象可被移动
        • wait：此光标指示程序正忙（通常是一只表或沙漏）
        • help：此光标指示可用的帮助（通常是一个问号或一个气球）
        • text：此光标指示文本
        • x-resize：显示双向箭头，x是其中一个箭头需指向的方向，可以是 n、nw
    • 边框
      • border
        • 简记法，同时设置四个边框的 border-style、border-width、border-color 属性
        • 如果需要，输入 -top、-right、-bottom、-left 将边框效果限制在某一条边上
          • border-top
            • 简记法，同时设置上边框的 border-style、border-width、border-color 属性
            • border-top-style
            • border-top-width
            • border-top-color
          • border-right
            • 简记法，同时设置右边框的 border-style、border-width、border-color 属性
            • border-right-style
            • border-right-width
            • border-right-color
          • border-bottom
            • 简记法，同时设置下边框的 border-style、border-width、border-color 属性
            • border-bottom-style
            • border-bottom-width
            • border-bottom-color
          • border-left
            • 简记法，同时设置左边框的 border-style、border-width、border-color 属性
            • border-left-style
            • border-left-width
            • border-left-color
    • 边框风格
      • border-style
        • 简记法，同时设置四个边框的风格属性。如果需要，输入 -top、-right、-bottom、-left 将边框效果限制在某一条边上
          • border-top-style
          • border-right-style
          • border-bottom-style
          • border-left-style
        • none：不显示边框。默认值。
        • dotted：点线
        • dashed：虚线
        • solid：实线
        • double：双线
        • groove：槽线
        • ridge：脊线
        • inset：凹边
        • outset：凸边
    • 边框宽度
      • border-width
        • 简记法，同时设置四个边框的宽度属性。如果需要，输入 -top、-right、-bottom、-left 将边框效果限制在某一条边上
          • border-top-width
          • border-right-width
          • border-bottom-width
          • border-left-width
        • medium：定义中等的边框。默认值。
        • thin：定义细的边框
        • thick：定义粗的边框
        • length：允许自定义边框的宽度
    • 边框颜色
      • border-color
        • 简记法，同时设置四个边框的颜色属性。如果需要，输入 -top、-right、-bottom、-left 将边框效果限制在某一条边上
          • border-top-color
          • border-right-color
          • border-bottom-color
          • border-left-color
        • transparent：指定边框的颜色应该是透明的。默认值。
        • colo：指定背景颜色
    • 边框图像
      • border-image
        • 简记法，同时设置四个边框的 border-image-source、border-image-slice、 border-image-width、border-image-outset、border-image-repeat 属性
    • 边框图像位置
      • border-image-source
        • none：没有图像被使用。默认值。
        • image：边框使用图像的路径
    • 边框图像边界向内偏移
      • border-image-slice
    • 边框图像边界宽度
      • border-image-width
    • 边框图像越界大小
      • border-image-outset
    • 边框图像填充
      • border-image-repea
        • stretch：拉伸图像来填充区域。默认值。
        • repeat：平铺（repeated）图像来填充区域
        • round：类似 repeat 值。如果无法完整平铺所有图像，则对图像进行缩放以适应区域
        • space：类似 repeat 值。如果无法完整平铺所有图像，扩展空间会分布在图像周围
    • 背景
      • background
        • 简记法，同时设置 background-color、background-image、background-size、background-repeat、background-position、background-origin、background-attachment、background-chip 属性
    • 背景颜色
      • background-color
    • 背景图像
      • background-image
    • 背景图像重复
      • background-repeat
        • repeat：背景图像将向垂直和水平方向重复。这是默认
        • repeat-x：只有水平位置会重复背景图像
        • repeat-y：只有垂直位置会重复背景图像
        • no-repeat：不重复
    • 背景图像滚动
      • background-attachment
        • scroll：背景图片随着页面的滚动而滚动，这是默认的
        • fixed：背景图片不会随着页面的滚动而滚动
        • local：背景图片会随着元素内容的滚动而滚动
    • 背景图像位置
      • background-position
        • left top
        • left center
        • left bottom
        • right top
        • right center
        • right bottom
        • center top
        • center center
        • center bottom
        • x% y%
          • 第一个值是水平位置，第二个值是垂直。左上角是0％0％。右下角是100％100％。如果仅指定了一个值，其他值将是50％。 。默认值为：0％0％
        • xpos ypos
          • 第一个值是水平位置，第二个值是垂直。左上角是0。

  • 文本

    • 字体
      • font
        • 简记法，同时设置字体 font-style、font-weight、font-size、font-height、font-family 属性
        • 必须始终显式地声明文本大小和字体属性：先是字体大小，再是字体系列
        • 行高是可选的，但它如果出现就必须紧跟在字体大小和斜杠后面
    • 类型
      • font-family
    • 斜体
      • font-style
        • normal：浏览器显示一个标准的字体样式，默认值
        • italic：浏览器会显示一个斜体的字体样式
        • oblique；浏览器会显示一个倾斜的字体样式
    • 粗细
      • font-weight
        • normal：定义标准的字符。默认值。
        • bold：定义粗体字符，常用
        • bolder：定义更粗的字符
        • lighter：定义更细的字符
        • 100-900：定义由细到粗的字符。400 等同于 normal，而 700 等同于 bold
    • 大小
      • font-size
        • xx-small
        • x-small
        • small
        • medium：默认值
        • large
        • x-large
        • xx-large
        • smaller：设置为比父元素更小的尺寸
        • larger：设置为比父元素更大的尺寸
        • length：设置为一个固定的值
    • 行高
      • line-height
        • normal：设置合理的行间距。默认。
        • number：设置数字，此数字会与当前的字体尺寸相乘来设置行间距
        • length：设置固定的行间距
    • 颜色
      • color
    • 字间距
      • word-spacing
    • 字偶距
      • letter-spacing
    • 首行缩进
      • text-indent
    • 单词内断行
      • word-break
        • normal：使用浏览器默认的换行规则
        • break-all：可在任意字符间换行，允许在单词内换行
        • keep-all：文本不断行，只能在半角空格或连字符处换行
    • 大小写
      • text-transform
        • none：定义带有小写字母和大写字母的标准的文本。默认
        • capitalize：文本中的每个单词以大写字母开头
        • uppercase：定义仅有大写字母
        • lowercase：定义仅有小写字母
    • 小型大写
      • font-variant
        • normal：浏览器会显示一个标准的字体。默认值
        • small-caps：浏览器会显示小型大写字母的字体。并非所有的字体都有对应小型大写字母设计
    • 文本线条
      • text-decoration
        • none
          • 定义标准的文本。默认。
        • underline
          • 下划线
        • overline
          • 上划线
        • line-through
          • 删除线
        • blink
          • 闪烁文本
        • 除了 a 元素外，也可以应用到其他元素
      • text-decoration-style
        • solid
          • 线条将显示为单线。默认值。
        • double
          • 线条将显示为双线
        • dotted
          • 线条将显示为点状线
        • dashed
          • 线条将显示为虚线
        • wavy
          • 线条将显示为波浪线
      • text-decoration-color
    • 空白属性
      • white-space
        • 指定元素内的空白怎样处理
        • normal
          • 空白会被浏览器忽略。默认值。
        • pre
          • 空白会被浏览器保留。类似 HTML 中 
             标签
        • nowrap
          • 文本不会换行，文本会在在同一行上继续直到遇到
            标签
        • pre-wrap
          • 保留空白符序列，但是正常地进行换行
        • pre-line
          • 合并空白符序列，但是保留换行符

  • 表格

    • 合并边框
      • border-collapse
        • separate：边框会被分开。不会忽略 border-spacing 和 empty-cells 属性。默认值。
        • collapse：如果可能边框会合并为一个单一边框。忽略 border-spacing 和 empty-cells 属性
    • 边框间距
      • border-spacing
        • 定义一个 length 参数，那么定义的是水平和垂直间距
        • 如果定义两个 length 参数，那么第一个设置水平间距，而第二个设置垂直间距

  • 增强

    • 创建圆角
      • border-radius: r
        • border-top-left-radius
        • border-top-right -radius
        • border-bottom-right-radius
        • border-bottom-left-radius
    • 创建椭圆形圆角
      • border-radius: x/y
    • 创建圆形
      • border-radius: r
    • 旋转缩放
      • transform
        • matrix(n,n,n,n,n,n)：定义 2D 转换，使用 6 个值的矩阵
        • matrix3d(n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n)：定义 3D 转换，使用 16 个值的 4x4 矩阵
        • translate(x,y)：定义 2D 转换
        • translate3d(x,y,z)：定义 3D 转换
        • translateX(x)：定义转换，只是用 X 轴的值
        • translateY(y)：定义转换，只是用 Y 轴的值
        • translateZ(z)：定义 3D 转换，只是用 Z 轴的值
        • scale(x[,y]?)：定义 2D 缩放转换
        • scale3d(x,y,z)：定义 3D 缩放转换
        • scaleX(x)：通过设置 X 轴的值来定义缩放转换
        • scaleY(y)：通过设置 Y 轴的值来定义缩放转换
        • scaleZ(z)：通过设置 Z 轴的值来定义 3D 缩放转换
        • rotate(angle)：定义 2D 旋转，在参数中规定角度
        • rotate3d(x,y,z,angle)：定义 3D 旋转
        • rotateX(angle)：定义沿着 X 轴的 3D 旋转
        • rotateY(angle)：定义沿着 Y 轴的 3D 旋转
        • rotateZ(angle)：定义沿着 Z 轴的 3D 旋转
        • skew(x-angle,y-angle)：定义沿着 X 和 Y 轴的 2D 倾斜转换
        • skewX(angle)：定义沿着 X 轴的 2D 倾斜转换
        • skewY(angle)：定义沿着 Y 轴的 2D 倾斜转换
        • perspective(n)：为 3D 转换元素定义透视视图
    • 不透明度
      • opacity
        • 默认值为 1 。当 opacity 属性的值应用于某个元素上时，是把这个元素（包括它的内容）当成一个整体看待，即使这个值没有被子元素继承。因此，一个元素和它包含的子元素都会具有和元素背景相同的透明度，哪怕这个元素和它的子元素有不同的 opacity 属性值
        • 使用 opacity 属性，当属性值不为 1 时，会把元素放置在一个新的层叠上下文中

  • 媒体

    • 渲染媒体
      • object-fit
        • 两个概念要分清楚：盒子（由元素的宽和高决定）和渲染的媒体。默认情况二者大小相等
        • object-fit 属性让我们能在盒子内部控制渲染媒体的大小，同时又保持盒子的大小不变，即元素的内容应该如何适应指定容器的高度与宽度，元素主要用于图片和视频
        • fill
          • 不保证保持原有的比例，内容拉伸填充整个内容容器。默认值。
        • none
          • 保留原有元素内容的长度和宽度，即不被重置
        • cover
          • 保持原有尺寸比例，让它填满盒子（导致图片一部分被裁剪）
        • contain
          • 保持原有尺寸比例，让它完整地填充盒子（导致盒子里出现空白）
        • scale-down
          • 保持原有尺寸比例。内容的尺寸是 none 或 contain 中的一个相同，取决于它们两个之间谁得到的对象尺寸会更小一些
